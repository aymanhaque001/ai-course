    🧠 Complete AI & Machine Learning Course¶

    From mathematical foundations to production AI systems — a comprehensive, self-paced curriculum for mastering artificial intelligence, with interview preparation at every level.

    Course Overview¶

    This course covers the full AI/ML stack — from linear algebra and classical machine learning through deep learning, transformers, LLMs, generative AI, and production systems. Every module includes:

    • Conceptual explanations with ASCII diagrams and comparison tables
    • Code implementations from scratch (Python/NumPy)
    • Interview questions — conceptual, coding, and system design
    • Key papers with context on why they matter

    # Module Topics Est. Time
    11 Mathematical Foundations Linear algebra, calculus, probability, information theory, optimization 8-10 hrs

    Core LLM Stack¶

    # Module Topics Est. Time
    1 Neural Network Foundations Neurons, activations, backprop, optimization, regularization, weight init 8-10 hrs
    2 Transformer Architecture Self-attention, multi-head, positional encoding, KV-cache, Flash Attention 8-10 hrs
    3 Tokenization & Embeddings BPE, tokenizers, Word2Vec, contextual embeddings, vector spaces 6-8 hrs
    4 Training LLMs Pre-training, SFT, LoRA/QLoRA, RLHF, DPO, alignment 8-10 hrs
    5 Prompt Engineering Few-shot, CoT, ReAct, prompt injection, DSPy, sampling 6-8 hrs
    6 Retrieval-Augmented Generation RAG architecture, chunking, vector DBs, hybrid search, Graph RAG 8-10 hrs
    7 Evaluation & Benchmarks Metrics, benchmarks, human eval, LLM-as-judge, hallucination 6-8 hrs
    8 Scaling & Inference Scaling laws, quantization, Flash Attention, speculative decoding, MoE 8-10 hrs
    9 Safety & Alignment HHH framework, harm taxonomy, bias, red teaming, governance 6-8 hrs
    10 Advanced Topics Agents, multi-agent systems, long-context, multimodal, mechanistic interp 8-10 hrs

    How to Use This Course¶

    1. Pick a learning path above based on your background
    2. Read each module sequentially — concepts build on each other
    3. Implement the code examples — don't just read them
    4. Answer the interview questions before revealing solutions
    5. Read the key papers (at least the abstracts and intro sections)
    6. Build something after every 3-4 modules to solidify understanding

    Total modules: 16 | Total interview questions: 100+ | Key papers: 80+
